# Erstellen von Aufgabengebieten und Tarifkarten

In diesem Video wird beschrieben, wie Sie ein Aufgabengebiet und eine Tarifkarte erstellen und wie Sie eine Tarifkarte einem Projekt anhängen und daraus entfernen können.


>[!VIDEO](https://video.tv.adobe.com/v/3486517/?quality=12&learn=on&enablevpops=1)

## Wichtige Schlussfolgerungen

* **Aufgabengebiete strukturieren Projektaufgaben:** Aufgabengebiete repräsentieren die für Aufgaben erforderlichen Fähigkeiten und können Benutzern auf Grundlage ihrer Fähigkeiten und Verfügbarkeit zugewiesen werden.
* **Tarifkarten definieren Abrechnungssätze:** Tarifkarten listen Abrechnungssätze für Aufgabengebiete auf und fungieren als Verträge mit Kunden für die Projektabrechnung.
* **Flexibles Tarifmanagement** Sowohl Aufgabengebiete als auch Tarifkarten ermöglichen das Festlegen von Kosten- und Abrechnungssätzen mit effektiven Datumsangaben, wodurch zeitbasierte Preisänderungen unterstützt werden.
* **Tarifkartenüberschreibungen:** Wenn eine Tarifkarte mit einem Projekt verbunden ist, überschreiben ihre Tarife die Tarife für Aufgabengebiete, projektspezifische Überschreibungen bleiben jedoch unberührt.
* **Einfaches Anhängen und Aktualisieren:** Tarifkarten können an Projekte angehängt oder aus Projekten entfernt werden, und Aktualisierungen können an vorhandene Projekte übertragen werden, um eine genaue Abrechnung sicherzustellen.
