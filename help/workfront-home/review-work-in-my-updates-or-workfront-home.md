# Überprüfung der Arbeiten in [!DNL My Updates] oder [!DNL Home]

Workfront bietet zwei Stellen, an denen Sie auf die Ihnen zur Überprüfung und Genehmigung zugewiesenen Arbeiten zugreifen können:

* [!DNL My Updates]
* [!DNL Home]

Auf der [!DNL My Updates] Seite werden alle Ihnen zugewiesenen Genehmigungsanfragen sowie Aktualisierungen, in denen Sie enthalten sind, angezeigt. Diese Seite wird für Benutzer der Überprüfungslizenz bei der Anmeldung bei Workfront standardmäßig geladen, es sei denn, der Systemadministrator Ihres Unternehmens hat eine andere Landingpage festgelegt.

![Ein Bild der Seite [!DNL My Updates]](assets/my-updates-overview.png)

[!DNL Home] kann für mehr als nur Genehmigungen verwendet werden. Wenn Sie eine Arbeits- oder Planlizenz in Workfront haben, finden Sie Ihre Zuweisungen aus Projekten und Anfragen auf der Startseite. Die Startseite kann die Seite sein, auf der Sie landen, wenn Sie sich bei Workfront anmelden. Alternativ können Sie auch Startseite aus dem Hauptmenü auswählen, je nach den Workfront-Systemkonfigurationen Ihres Unternehmens.

![Ein Bild der Seite [!DNL Home]](assets/home-overview.png)

Welcher Abschnitt Sie für die Verwaltung Ihrer Genehmigungsanfragen verwenden, hängt von den Prozessen Ihres Unternehmens zur Überprüfung und Genehmigung Ihrer Arbeit ab. Wenden Sie sich an Ihre Manager oder Managerinnen, Systemadmins oder das interne Workfront-Team, um zu erfahren, welches System Sie verwenden sollten, um Ihren Workflow optimal zu unterstützen.
